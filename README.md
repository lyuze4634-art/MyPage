# MyPage
给自己用的简历个人页面，上传到服务器了，网址是itsnotabug.cc
上传服务器的时候查一下自己的服务器有没有fileinfo组件，没有的话就要绕开mime_content_type这个玩意。
最基础的增删改查都有，之后计划加入一些有趣的内容，比如写一个自动扣图之类的玩意放上去。
或者接入Ai训练出一个赛博李雨泽，我哪天要是死了可以和赛博李雨泽聊天。

自分用の履歴書個人ページを作って、すでにサーバーにアップロードしました。URL は itsnotabug.cc です。
サーバーにアップロードする際は、自分のサーバーに fileinfo 拡張が入っているか確認したほうがいいです。入っていない場合は、mime_content_type というやつを回避する必要があります。
一番基本的な CRUD は一通り実装していて、今後はもっと面白い内容も追加する予定です。例えば、自動で背景を切り抜くようなものを書いて載せるつもりです。
あるいは AI を接続して「サイバー李雨泽」を育てるのも考えています。いつか私が死んだとしても、そのサイバー李雨泽 と話せるようにするためです。
